はじめに

皆さんはプログラミングをしたことや興味を持ったことがありますか？
初めてプログラミングに触れる方の中にはプログラミングやC言語という言葉自体は知っていても
実際にどのようなものか具体的なイメージがわかない方も多いのではないのでしょうか。
本稿では代表的なプログラミング言語であるC言語を題材として、
初めてプログラミングを経験する人に向けて解説していきたいと思います。

C言語の特徴

まず、今回解説していくC言語についてざっくりと説明していきたいと思います。
　C言語は他の多くのプログラミング言語の祖先といえる存在で、
少々煩雑な一方でプログラムの仕組みや基礎を学び、理解するために最適な言語です。
　プログラミング言語によって書かれる文章を「コード」と言いますが、
コードを機械が読んで実行する際にC言語は機械にとってシンプルで理解しやすい構造をしています。
　そのため機械による読み込みや実行がスムーズに行える利点があり、それらの利点を生かして現在でも幅広く活用されています。
　先に書いたようにC言語は他の言語の祖先と言える存在であるためC言語で学んだことはほかの言語でも応用が利くことがあり、
いわば語学学習の英語に相当する言語です。
（難易度を除けば）初めて学ぶプログラミング言語としてC言語は適していると言えますね。

言語の基本　Helloworld


以下に示すのがプログラミング言語の基本、実行するとHelloworldと文字列が表示されるプログラムです。

#include

int main() {
printf("Hello, World!\n");
return 0;
}


今はこれだけ見てもどのような仕組みになっているかわからないと思うかもしれませんが、
ここからの解説を通してHelloworldに含まれる各要素の意味を少しずつ理解してもらえたらなと思います。
また、初めの#include 　と最後の　return 0;　と　int(後に解説する)char floatでも可能　main(){プログラムの中身}
はプログラムの初めと終わりを定義してまとめるために必要な物なので、
欠かさず入力するようにしてください。
本稿全体ではこれらの定型部分を省略したコードを例として使用しているのでご注意ください。

早速実践 基本変数（int char float）


ここからはプログラミングをするうえで欠かせない変数について解説していきます。
変数とはプログラムの中でデータを一時的に保存しておくための「名前が付いた箱」のようなもので、
変数の種類が箱の名前です。

変数を使うには何という変数を使うかの「宣言」を行い必要に応じて値を代入することになります。
↓例
int ; level // 変数を宣言
level = 18; // 値を代入

float price = 12.5; // 宣言と同時に代入

ちなみにC言語ではコードに影響を及ぼさない文字列をメモなどの用途で書くために
コメントという機能があり、//の後に文字を書くことで使えます。

以下に代表的な変数をゲームのプログラムで使うと仮定した例を交えて紹介します。

１．int　（整数型）
整数を格納するための変数です。

例　int level = 18;　//レベルを１８とする

このようにプレイヤーのレベル等の整数の値を定義する際に使われます。

２．float　(浮動小数点型) 少数を含む値を格納するための変数です。

例　float price = 12.5 ; //値段を12.5とする

このように物の値段などを定義する際に利用します。ほとんど小数点があるか否かだけの違いなのでintに近い使われ方をします。

３．char (文字型)
１文字を格納するための変数です。

例　char grade = 'A'； //グレードをAとする

アイテムのグレードを表したり,１文字で表現できる概念の定義に使われます。

このような関数が実際にゲームのプログラムなどでも使われています。
また、最初に示したHelloworldの　int main()
この部分のintはその先に書かれるコードで使用される変数を定義するために入れられます。
そのためコードの形態に応じてint char float等の中から適したものを選択して定義する必要があります。
　余談ですが、char型を応用すれば文字列を格納することができます。

例　char　playerName[20]　; //プレイヤー名のメモリを最大20文字で確保

このようにしてメモリを確保してそのメモリに実際のプレイヤー名を代入することで複数文字、
言い換えれば文字列を格納することができます。


文字の入出力



　次に実際にコードを書く中で特に重要になってくる「文字を入出力する際に使う関数」について解説します。
　
１．printf関数
これは指定した形式でテキストや変数の値などの情報を画面に出力する関数です。

基本構造としては　printf("表示する文章 \n"); のように扱います。

２．scanf関数
こちらはprintf関数の逆ともいえるもので、ユーザーが入力した文字列を読み取る命令を行うものです。

基本構造としては　scanf("あなたの名前を教えてください ￥n")
のように扱います。

文字列を表示したり読み取る際には文字列の末尾に\nを入れて””で囲むことを忘れないようにしましょう。
また、最後に入っている；は改行を意味するので、これもこまめに入れるようにしましょう。

四則演算



　次に解説するのはコンピューターが最も得意とする計算です。
　四則演算を利用することでゲームでは経験値や所持金を加算していったり、
攻撃のダメージ値を求めたり様々なことができます。

　C言語での四則演算はシンプルで、
足し算は＋　引き算は-　掛け算は* 割り算は/で表すことができます。

　ただ、実際にプログラム上で計算する場合多くは計算に使用する値を読み取ってそれを関数に代入して計算します。
ここでは足し算を用いた例を示します。そのほかの-*/も基本的には同じです。

printf ("値１を指定してください");　//値１を指定してもらうよう表示
scanf ("%f , num1 ");　//入力された値を読み取りnum1と定義
printf ("値２を指定してください");　//値２を指定してもらうよう表示
scanf ("%f,num2");　//入力された値を読み取りnum2と定義
printf("足し算"　, num1 + num2);　//足し算　の文字と計算された値を表示このようにして四則演算を行うことができます。
今回の例では前項で取り上げたprintfとscanfも取り入れた物にしました。


if文とfor文


次にプログラミング言語を使う上でかなり重要になってくるifとforについて解説します。
if文は特定の条件が満たされた場合にだけ処理を実行するための構文です。
一方でfor文は何かしらの処理を繰り返し実行するための構文です。

どちらもループ処理に使われることがありますが、ifは条件分岐forは単純な繰り返しとして捉えると分かりやすいです。
具体的にはif文だとオート戦闘で敵のhpがなくなるまで攻撃を繰り返す、
for文だと敵のhpの有無にかかわらず決められた回数攻撃を繰り返すこのような違いが生まれてしまうため、使い分けが重要です。

if文の例

printf("成績点を入力してください: "); //スコアをユーザーにたずねる
scanf("%d", &score); //ユーザーの入力した値を読み取る
if(score >= 60) {
printf("おめでとう単位認定！\n"); //スコアが60以上なら、おめでとう単位認定！と表示
　　} else{
printf("やったね落単！\n"); //もしそうでないなら　やったね落単！と表示
}

高専生がみると発狂しそうな例ですが、このように値が基準値を満たしているか否かの判別などが行えます。
ちなみにelseはそれ以外を意味する英単語で
条件に当てはまらない場合を出力する際に使います。

for文の例
for(inti = 1; i ＜= 100; i++) {
printf("おはようございます！％ｄ回目の目覚ましだぞ流石に起きろ！\n");
}

100回まで目覚ましのテキストを表示して流石に起きろと脅すプログラムでした。
もちろんテキストのみだと目覚ましとして機能しないので
実際に目覚ましのプログラムでは音を鳴らす命令を別途組み込む必要があります。
